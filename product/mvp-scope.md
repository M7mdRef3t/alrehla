# MVP Scope — الرحلة (أداة دواير)

## MVP Goal
المستخدم يرسم خريطة علاقاته بسرعة داخل أداة **دواير**، ويحصل على "قرار عملي" بسيط بدون تنظير.

## MVP Screens (First Release)
1) Landing
2) Goal Picker (اختيار النية)
3) Core Map (Drag & Drop)
4) Feeling Check (3 أسئلة)
5) Result + Action Toolkit
6) Emergency Button

## Out of Scope (Later)
- حسابات/تسجيل
- مجتمع/مشاركة
- توصيات معقدة
- تحليلات طويلة
