Place production-ready screenshots in this folder.

- 1200x628 (OG / large preview)
- 800x600 (gallery)
- 640x360 (mobile preview)

Filenames expected by manifest.json:
- hero-1.png  -> /screenshots/hero-1.png  (1200x628)
- hero-2.png  -> /screenshots/hero-2.png  (800x600)
- hero-3.png  -> /screenshots/hero-3.png  (640x360)

Notes:
- Use high quality PNG or WebP.
- Keep UI chrome minimal and include readable Arabic UI text.
- After adding images, commit them and verify `public/manifest.json` references the correct paths.
