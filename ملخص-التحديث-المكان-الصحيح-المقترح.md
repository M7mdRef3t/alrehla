## 🎯 قبل وبعد

### قبل:
```
النتيجة: "قربك من ماما مؤلم ومحتاج حماية"
الحالة: استنزاف (دائرة حمراء)

❌ المشكلة: المستخدم عارف المشكلة، بس مش عارف الحل
```

### بعد:
```
النتيجة: "قربك من ماما مؤلم ومحتاج حماية"

المكان الحالي: 🚨 دائرة الاستنزاف (أحمر)
         ↓
المكان الصحيح المقترح: ⚠️ دائرة القرب المشروط (أصفر)

ليه؟ "العلاقة دي ليها دور في حياتك، بس محتاج حدود واضحة..."

إزاي توصل للمكان الصحيح:
1. حدد أوقات محددة للتواصل (مثلاً: مكالمة واحدة في الأسبوع)
2. ما تردش على كل مكالمة فوراً
3. لو حصل استنزاف، اعتذر وانسحب
...

✅ الحل: المستخدم عنده roadmap واضح
```

## 📚 منطق تحديد المكان المقترح

### الحالة 1: العلاقة في الدائرة الخضراء ✅
```
المكان الحالي: أخضر
المكان المقترح: أخضر (استمر كده!)

الرسالة: "العلاقة في مكانها الصحيح!"
الخطوات: حافظ عليها
```

---

### الحالة 2: العلاقة في الدائرة الصفراء ⚠️

#### لو الأعراض كتيرة (5+):
```
المكان الحالي: أصفر
المكان المقترح: أصفر (محتاج حدود أقوى)

السبب: "العلاقة محتاجة حدود أوضح عشان ما تنزلقش للحمراء"

الخطوات:
- حدد أوقات واضحة للتواصل
- تعلم تقول 'لأ' بدون ذنب
- راقب الأعراض
```

#### لو الأعراض قليلة (1-4):
```
المكان الحالي: أصفر
المكان المقترح: أخضر (ممكن تتحسن!)

السبب: "العلاقة ممكن تتحسن وتبقى صحية لو اتعملت تعديلات بسيطة"

الخطوات:
- وضّح توقعاتك
- اتكلم عن اللي بيضايقك
- ادي فرصة للتغيير
```

---

### الحالة 3: العلاقة في الدائرة الحمراء 🚨

المكان المقترح **يعتمد على نوع العلاقة:**

#### العائلة (أم، أب، أخ/أخت):
```
المكان الحالي: أحمر
المكان المقترح: أصفر (مع حدود قوية)

السبب: "العلاقة دي ليها دور في حياتك، بس محتاج حدود واضحة"

الخطوات:
1. حدد أوقات محددة للتواصل (مكالمة واحدة/أسبوع، 30 دقيقة)
2. ما تردش فوراً - خد وقتك
3. لو حصل استنزاف، اعتذر وانسحب
4. ما تشاركش كل تفاصيل حياتك
5. خليك حازم - التنازل بيرجعك للأحمر
```

#### الشريك الرومانسي/الزوج:
```
المكان الحالي: أحمر
المكان المقترح: أصفر (آخر فرصة)

السبب: "العلاقة محتاجة مسافة أكبر + حدود واضحة"

الخطوات:
1. قلل التواصل لأقل حد
2. مكالمات قصيرة (15 دقيقة max)
3. أماكن عامة فقط
4. متشاركش حاجة شخصية
5. لو الاستنزاف زاد → قطع تام
```

#### الصديق/الزميل:
```
المكان الحالي: أحمر
المكان المقترح: أصفر (أو قطع)

السبب: "العلاقة محتاجة مسافة كبيرة"

الخطوات:
1. قلل التواصل (مرة كل أسبوعين أو أقل)
2. مكالمات قصيرة جداً
3. أماكن عامة فقط
4. صفر تفاصيل شخصية
5. فكر جدياً في القطع التام
```

#### الـ Ex:
```
المكان الحالي: أحمر
المكان المقترح: أصفر (أو قطع تام)

السبب: "الأفضل قطع أو مسافة كبيرة جداً"
```

## 📊 أمثلة واقعية

### مثال 1: الأم في الدائرة الحمراء

**المدخلات:**
- الشخص: ماما
- Ring: أحمر
- الأعراض: ذنب + إرهاق + تلاعب عاطفي (3 أعراض)

**المخرجات:**
```
المكان الحالي: 🚨 دائرة الاستنزاف (أحمر)
المكان المقترح: ⚠️ دائرة القرب المشروط (أصفر)

ليه؟
"العلاقة دي ليها دور في حياتك، بس محتاج حدود واضحة عشان 
ما تستنزفكش. المكان الصحيح: دائرة القرب المشروط (الأصفر) - 
تواصل محدود + حماية نفسية."

إزاي توصل للمكان الصحيح مع ماما:
1. حدد أوقات محددة للتواصل (مثلاً: مكالمة واحدة في الأسبوع، 30 دقيقة)
2. ما تردش على كل مكالمة/رسالة فوراً - خد وقتك
3. لو حصل استنزاف في اللقاء، اعتذر بهدوء وانسحب
4. ما تشاركش كل تفاصيل حياتك - احتفظ بمساحة ليك
5. خليك حازم على الحدود - التنازل بيخليك ترجع للدائرة الحمراء

⚠️ مهم: التغيير محتاج وقت وثبات. خليك ملتزم بالحدود اللي 
هتحطها - أي تنازل ممكن يرجعك للدائرة الحمراء تاني.
```

---

### مثال 2: صديق سام في الدائرة الحمراء

**المدخلات:**
- الشخص: أحمد (صديق)
- Ring: أحمر
- الأعراض: تفكير دائري + إهمال نفس + تجنب (3 أعراض)

**المخرجات:**
```
المكان الحالي: 🚨 دائرة الاستنزاف (أحمر)
المكان المقترح: ⚠️ دائرة القرب المشروط (أصفر)

ليه؟
"العلاقة دي محتاجة مسافة أكبر + حدود واضحة. المكان الصحيح: 
دائرة القرب المشروط (الأصفر) - تواصل محدود ومشروط بالاحترام 
المتبادل."

إزاي توصل للمكان الصحيح مع أحمد:
1. قلل التواصل لأقل حد ممكن (مرة كل أسبوعين أو أقل)
2. لو محتاج تتواصل، خليها مكالمات قصيرة (15 دقيقة max)
3. ما تلتقيش في أماكن خاصة - أماكن عامة أفضل
4. متشاركش أي حاجة شخصية عن حياتك
5. لو زاد الاستنزاف، ابدأ فكر في القطع التام
```

```
• حافظ على التواصل المنتظم

• استمر في احترام الحدود المتبادلة
• قدّر وجود الشخص ده في حياتك
```

## 🎯 الفوائد

### 1. Clarity (وضوح)
- المستخدم مش بس عارف المشكلة
- عارف **الحل المقترح** كمان

### 2. Actionable Roadmap
- خطوات محددة
- مش نظري - عملي 100%

- العائلة ≠ الأصدقاء ≠ الشركاء

- مش بنقول "اقطع العلاقة مع أمك"
- بنقول "حط حدود واضحة"
- Balanced approach

- "⚠️ مهم: التغيير محتاج وقت..."
- يحذر من التنازل
- يوضح العواقب
