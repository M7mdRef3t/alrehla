# Investor Data Room Checklist

## A) Company & Legal
- [ ] Company registration docs
- [ ] Cap table (latest + previous)
- [ ] Founder agreements
- [ ] IP assignment agreements
- [ ] Privacy policy / terms

## B) Product
- [ ] Product vision (1 pager)
- [ ] Roadmap (90-day + 12-month)
- [ ] Core architecture summary
- [ ] Security posture summary
- [ ] Reliability/SLA plan

## C) Traction & Metrics
- [ ] Last 6 months KPI table
- [ ] Cohort retention analysis
- [ ] Activation funnel trends
- [ ] Channel attribution report
- [ ] Churn breakdown

## D) Financials
- [ ] P&L (monthly, last 12 months)
- [ ] Cash flow
- [ ] Burn & runway model
- [ ] Pricing model assumptions
- [ ] Revenue forecast scenarios (base/upside/downside)

## E) Go-To-Market
- [ ] ICP definition
- [ ] Channel strategy
- [ ] Sales motion / conversion cycle
- [ ] CAC by channel
- [ ] Expansion strategy

## F) Team
- [ ] Org chart
- [ ] Hiring plan (next 12 months)
- [ ] Key role gaps
- [ ] Advisor list

## G) Risk Register
- [ ] Top 10 product risks + mitigations
- [ ] Top 10 security risks + mitigations
- [ ] Operational risks + monitoring
- [ ] Dependency risks (infra, vendors, APIs)

## H) Investor Narrative (ready-to-send)
- [ ] Problem
- [ ] Why now
- [ ] Solution
- [ ] Traction proof
- [ ] Moat
- [ ] Use of funds

## Minimum bar before fundraising outreach
- [ ] Metrics definitions locked
- [ ] Weekly investor memo cadence active >= 6 weeks
- [ ] Data quality checks pass
- [ ] Security/perf gates stable in CI
