## Purpose

هذا الملف هو قالب المراجعة الأسبوعية للأونر.

يُستخدم مرة كل أسبوع لمراجعة:

- حالة المنصة
- المسارات الأساسية
- الانحرافات
- القرارات القادمة

الهدف ليس كتابة تقرير طويل، بل اتخاذ قرار أوضح كل أسبوع.

يرتبط هذا الملف مع:

- `docs/owner-ops-manual.md`
- `docs/critical-flows-checklist.md`
- `docs/feature-flag-matrix.md`
- `docs/post-release-verification-checklist.md`
