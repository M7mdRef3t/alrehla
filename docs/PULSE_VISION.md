# نظام النبض اللحظي — الرؤية والتنفيذ

> "الحالة اللحظية قبل كل شيء" — البوابة كترمومتر قبل الدخول للرحلة.

## 1. شاشة ضبط البوصلة (المنفّذ ✅)

| المطلوب | الحالة |
|--------|--------|
| Modal عند فتح التطبيق / عند "ابدأ الاَن" | ✅ يظهر عند start recovery وكل فتح (حسب `pulse_check` + `checkInMode`: daily / everyOpen) |
| مؤشر البطارية (فايق ↔ فاصل شحن) | ✅ سلايدر 1–10 في `PulseCheckModal` |
| الطقس الداخلي (رايق / هادئ / قلقان / غضبان / حزين) | ✅ أيقونات في `PulseCheckModal` |
| التركيز الحالي (موقف حصل، فكرة، جسدي، ولا حاجة) | ✅ خيارات في `PulseCheckModal` |
| مدة ~5 ثواني | ⚠️ المستخدم يضغط "جاهز" — لا يوجد تايمر أو اختيارات افتراضية لـ "سريعة جداً" |

## 2. التكيّف الديناميكي (المنفّذ)

| الحالة | الرؤية | المنفّذ حالياً |
|--------|--------|----------------|
| **أ: طاقة منخفضة** 🪫 | Dark Mode، إخفاء مهام المواجهة، زر "وضع الشرنقة" فقط، رسالة AI | ✅ Dark يُفعّل تلقائياً + إشعارات تُؤجّل 4 ساعات. ✅ زر "وضع الشرنقة" يظهر على الخريطة. ⚠️ الخريطة والمهام ما زالت ظاهرة (مافيش إخفاء كامل للمواجهة). ✅ الـ AI بيستلم النبض ويعدّل الرد (AIChatbot + pulseInfo). |
| **ب: غضبان** 🌩️ | واجهة تميل للأحمر، فتح "تفريغ الضجيج" فوراً، رسالة AI | ✅ أداة "إسكات الضجيج" **بتفتح تلقائياً** بعد الضغط على "جاهز". ✅ كارد وردي على الخريطة. ⚠️ لا تغيير ثيم عام لأحمر. |
| **ج: فايق** ⚡️ | ألوان زاهية، "تحدي اليوم" (أصعب مهمة) | ✅ كارد أخضر + زر "مناورة اليوم" (تحدي اليوم). ⚠️ لا تغيير ثيم عام لـ "زاهي". |

## 3. تقرير التقلبات (المنفّذ جزئياً)

| المطلوب | الحالة |
|--------|--------|
| بعد أسبوع: "طاقتك بتقل يوم X" | ✅ `getWeeklyPulseInsight`: يظهر "لاحظنا إن طاقتك بتقل غالبًا يوم [الثلاثاء]..." على الخريطة |
| ربط بـ "اليوم اللي عندك فيه اجتماع مع (فلان)" | ✅ منفّذ: `weekdayLabels` في pulseState + واجهة "ربط يوم بنشاط أو شخص" في كارد التقرير. التقرير يظهر "ده اليوم اللي عندك فيه [اللبل].. تعال نجهز له بشكل مختلف." |

## 4. فجوات مقترحة (اختياري)

1. **وضع "سريع" لضبط البوصلة:** اختيارات افتراضية أو تخطي بضغطة واحدة (مثلاً "كمل بدون تعديل") لتحقيق "5 ثواني".
2. ~~**وضع طاقة منخفضة**~~ — منفّذ: عند طاقة منخفضة تُخفى الخريطة والمهام وأزرار الإضافة؛ يظهر الهيدر + كارد الشرنقة فقط.
3. **ثيم حسب النبض (اختياري):** غضبان → لمسة حمراء للواجهة، فايق → لمسة زاهية (حالياً الكاردات فقط).
4. ~~**تقرير التقلبات المتقدم**~~ — منفّذ: ربط يوم بنشاط/شخص عبر `weekdayLabels` + مودال "ربط يوم بنشاط أو شخص" من كارد التقرير.

---

**الخلاصة:** البوابة (الترمومتر) والتحوّل حسب النبض (Low → Dark + شرنقة، Angry → إسكات الضجيج فوراً، High → تحدي اليوم) وتقرير الأسبوع الأساسي منفّذة. الباقي تحسينات UX وثيم وتقرير أعمق عند توفر البيانات.
