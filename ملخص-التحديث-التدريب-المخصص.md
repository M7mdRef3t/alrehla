    - التدريب على: تحويل التجنب لحدود

## 🎮 آلية التدريب

### 1. بنية كل سيناريو:
```typescript
{
  id: "guilt-scenario-1",
  symptomId: "guilt",
  title: "رفض طلب غير معقول",
  context: "الشخص اتصل بيك الساعة 11 بالليل...",
  situation: "قالك: 'محتاج أكلمك دلوقتي...'",
  options: [
    {
      id: "guilt-1-a",
      text: "طبعاً، اتفضل (وتسهر رغم تعبك)",
      isCorrect: false,
      feedback: "❌ استنزاف",
      explanation: "أنت ضحيت براحتك..."
    },
    {
      id: "guilt-1-c",
      text: "أنا تعبان دلوقتي، نتكلم بكرة الصبح؟",
      isCorrect: true,
      feedback: "✅ حدود صحية",
      explanation: "أنت احترمت احتياجك..."
    }
  ]
}
```

### 2. تدفق التدريب:
```
1. المستخدم يختار أعراض في شاشة النتيجة
2. في شاشة "أول خطوة" يظهر زر: "🎯 تدرب على التعامل مع [الشخص]"
3. يفتح PersonalizedTraining modal
4. يشوف سيناريوهات مخصصة بناءً على أعراضه
5. يختار إجابة → feedback فوري
6. في النهاية: تقييم + نصيحة شخصية
```

### 3. نظام التقييم:
```
- Perfect (100%): "أنت فاهم الحدود الصحية كويس جداً!"
- Good (70%+): "أنت على الطريق الصح!"
- Needs Work (<70%): "التدريب محتاج تكرار"
```
