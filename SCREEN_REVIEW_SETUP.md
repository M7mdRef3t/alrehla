## ⚠️ مشاكل UX المكتشفة

### 1️⃣ **وضوح الرسالة الرئيسية** ⚠️

**المشكلة:**
- العنوان "ضبط الوصلة" غير واضح للمستخدم الجديد
- النص "البحث والتحكم في الشؤون" غامق ومبهم
- لا يفهم المستخدم: هذا ماذا بالضبط؟

**الحل المقترح:**
```
قديم: "البحث والتحكم في الشؤون"
جديد: "اختر الأعراض المؤثرة الآن لبناء خطتك"
      أو
      "من أين تحتاج نبدأ معك؟"
```

### 2️⃣ **الأزرار والخيارات غير واضحة** ⚠️

**المشكلة:**
- أزرار "دقيق استشعار"، "عشاق"، "ملائم" (مترجمة غريبة/غير واضحة)
- "راجع"، "أحمر"، "أخضر" — عندما يعني "Red Ring" و "Green Ring"؟
- المستخدم لا يعرف كيف يختار أو ماذا تعني

**الحل المقترح:**
```
قديم: [دقيق استشعار] [عشاق] [ملائم]
جديد: [عائلة] [شغل] [حب] 
      (أو اسم السياق الصحيح)

قديم: [راجع] [أحمر] ✓ [أخضر] ✓
جديد: [تحذير] [🔴 خطر] [🟢 آمن] [🟡 مشروط]
      (مع أيقونات واضحة)
```

### 3️⃣ **Progress Bar غير منطقي** ⚠️

**المشكلة:**
- "5/50" — لماذا 50؟ هذا عرض غريب
- هل 50 هو عدد الخطوات؟ أم أسابيع؟
- لا يفهم المستخدم كم متبقي

**الحل المقترح:**
```
قديم: [5/50]
جديد: [الأسبوع الأول من 4 أسابيع] - 25%
      أو
      [5 خطوات من 12 خطوة] - 42%
```

### 4️⃣ **"مؤشرات فائقات"** ⚠️

**المشكلة:**
- هذا الاسم غريب ولا يعني شيء واضح
- هل هو "Super Indicators"؟
- ماذا يعني للمستخدم؟

**الحل المقترح:**
```
قديم: "مؤشرات فائقات ✓"
جديد: "المؤشرات الرئيسية ✓"
      أو
      "علامات حمراء نركز عليها ✓"
      أو
      "الأعراض المشروطة ✓"
```

### 5️⃣ **ترجمات غريبة في الأزرار** ⚠️

**المشكلة:**
- "عشاق" (بدل "شريك" أو "شخص")
- "دقيق استشعار" (بدل "شعور دقيق" أو "إحساس")
- "ملائم" (غير واضح السياق)
- "موقف جديد" (موقف ماذا؟)
- "درس منفتح" (هذا الجملة غير كاملة)

**الحل المقترح:**
```
قديم: "عشاق"      → جديد: "شريك"
قديم: "دقيق استشعار" → جديد: "حساس"
قديم: "ملائم"     → جديد: "مناسب"
قديم: "موقف جديد" → جديد: "موقف حديث" أو "حدث جديد"
قديم: "درس منفتح" → جديد: "درس مفتوح" أو "درس متقدم"
```

### 6️⃣ **أزرار العمل غير واضحة** ⚠️

**المشكلة:**
- "🟢 فك حدود الاتصال" — هل هذا ACTION أم STATUS؟
- "🟢 احفظ الدرية والحال" — "الدرية" كلمة غريبة، "الحال" غير واضح

**الحل المقترح:**
```
قديم: [🟢 فك حدود الاتصال]
جديد: [✓ تابع مع الحدود]
      أو
      [➜ الخطوة التالية]

قديم: [🟢 احفظ الدرية والحال]
جديد: [💾 احفظ الخطة]
      أو
      [✓ أكمل]
```

### 7️⃣ **المركز الحالي وقائمته غير واضحة** ⚠️

**المشكلة:**
- "المركز الحالي" وبجانبه X — هل يقصد "الموقع الحالي"؟
- القائمة تحته:
  - "موقف جديد"
  - "درس منفتح"
  - "سجسل تعالي"
  
  كل هذه غير واضحة وقد تكون ترجمات سيئة

**الحل المقترح:**
```
قديم: "المركز الحالي ✗"
جديد: "حالتك الحالية"
      أو
      "نقطة البداية"

القائمة:
  - "موقف جديد" → "حدث حديث"
  - "درس منفتح" → "تعلم جديد"
  - "سجسل تعالي" → ????? (هذا لا معنى له!)
```
