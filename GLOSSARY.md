# قاموس الرحلة — Journey Glossary

> المرجع الرسمي للغة المنصة. كل نص يواجه المسافر **يجب** يلتزم بالبدائل المذكورة هنا.
> أي مطور أو AI يكتب copy جديد، يرجع لهذا الملف أولاً.

---

## الكلمات الممنوعة والبدائل

| ❌ ممنوع في نصوص المسافر | ✅ البديل | السياق والاستخدام |
|--------------------------|----------|-------------------|
| أداة / أداتنا | بوصلة / مرآة | وصف ما تقدمه الرحلة |
| تطبيق / التطبيق | رحلتك | مخاطبة المستخدم |
| منصة / المنصة | الرحلة | وصف عام يواجه المسافر |
| منتجنا / خدمتنا | رحلتك / الرحلة | وصف ما نقدمه |
| مستخدم / مستخدمين | مسافر / مسافرين | وصف الشخص |
| خدمة (feature) | محطة في الرحلة | وصف ميزة |
| فريق عمل | رفاق الطريق | التواصل والتوقيع |
| عميل | مسافر | B2B أو عام |
| افتح التطبيق | ارجع لرحلتك | CTAs والإشعارات |
| استخدم المنصة | ابدأ رحلتك / كمّل رحلتك | CTAs |
| ابدأ التطبيق | ابدأ رحلتك | أول خطوة |
| يقفل المنصة | يغادر الرحلة | Analytics |

---

## استثناءات مسموحة

| الحالة | المصطلح المسموح | السبب |
|--------|----------------|-------|
| PWA Install | "تثبيت التطبيق" / "ثبّت على الهاتف" | action تقني |
| Admin Dashboard | "المنصة" / "أداة" | وصف للأونر |
| JSDoc/Comments | أي مصطلح تقني | كلام مطورين |
| API responses | "Not Found" / "Error" | HTTP standard |
| Code variables | `platformConfig`, `appEnv` | تسمية تقنية |

```
"مسافري الرحلة"
```
