# الخطوات القادمة المقترحة 🚀

بناءً على المناقشة، في أفكار إضافية كانت مطروحة. ده تلخيص لها مع تقييم الأولوية.

## المقترحات من المناقشة

### 1. شاشة "فهم أكتر" (Understanding Screen) - اختيارية

**الفكرة:**
شاشة إضافية بعد النتيجة تشرح **أنماط** العلاقة المستنزفة.

**مثال (للأم):**
```
🔍 ليه العلاقة مع "ماما" بتستنزفك؟

📌 أنماط العلاقة المستنزفة مع الأم:

1. الأم المتحكمة
   "مش هقدر أعيش من غيرك... عايزة تبعد عني؟"
   
2. الأم اللي محتاجة رعاية عاطفية
   "أنا تعبانة... أنت الوحيد اللي يفهمني"
   
3. الأم الناقدة دايماً
   "ليه مش عامل زي أخوك؟"

💡 النمط اللي انت فيه (بناءً على الأعراض):
"الأم المتحكمة + الذنب العاطفي"

إزاي بيحصل الاستنزاف:
- بتحس إنك مسؤول عن سعادتها
- أي قرار ليك بيبقى "أنانية"
- الذنب بيمنعك من وضع حدود

[كيفية التعامل ←] [ابدأ التعافي ←]
```

**التقييم:**
- ✅ **الإيجابيات**: فهم أعمق، يساعد المستخدم يحس إنه "مش لوحده"
- ⚠️ **السلبيات**: شاشة إضافية قد تطيل الرحلة
- 🎯 **الأولوية**: متوسطة (يمكن تأجيلها بعد MVP)

**التنفيذ المقترح:**
- نخليها **اختيارية** - زرار "فهم أكتر" في الشاشة الأولى
- من يضغط عليها يروح لشاشة التفصيل
- من ما يضغطش يكمل عادي

---

### 2. شاشة "كيفية التعامل" (Quick Tips) - اختيارية

**الفكرة:**
شاشة بنصائح سريعة **قبل** الدخول في الخطة الكاملة.

**مثال:**
```
🛠️ إزاي تتعامل مع "ماما" من غير استنزاف؟

خطوات فورية (ابدأ من النهاردة):

1️⃣ حدد وقت التواصل
   ✅ "أنا هكلمك يوم السبت الساعة 6"
   ❌ متسيبش الموضوع مفتوح

2️⃣ اتعلم تقول "لأ" بهدوء
   ✅ "مش هقدر دلوقتي... ممكن نشوف وقت تاني"
   ❌ متبررش كتير

3️⃣ متدخلش في دور "المنقذ"
   ✅ "أنا فاهمك، بس ده قرارك أنت"
   ❌ متحاولش تحل كل مشاكلها

[تدرب على الحدود ←] [خطة التعافي الكاملة ←]
```

**التقييم:**
- ✅ **الإيجابيات**: نصائح عملية فورية، immediate value
- ⚠️ **السلبيات**: overlap مع "الخطوة الأولى"
- 🎯 **الأولوية**: منخفضة (ممكن ندمجها في الخطوة الأولى)

### 3. شاشة "التدريب" (Simulation) - مهمة!

**الفكرة:**
سيناريوهات تفاعلية زي "جيم العلاقات" بس مخصصة للشخص.

**مثال:**
```
🎯 تدريب: ازاي تحط حدود مع "ماما"

موقف 1: ماما اتصلت الساعة 11 بالليل تشتكي
❓ هترد إزاي؟

A) أرد وأسمعها للآخر (✗ استنزاف)
B) ما أردش خالص (✗ ذنب)
C) "ماما، أنا هكلمك بكرة الصبح" (✓ حدود صحية)

[شرح] ليه C هي الأفضل:
- بتحترم وقتك
- بتحترمها (مش بتتجاهلها)
- بتدي بديل واضح

[التالي ←] [روح للخطة الكاملة ←]
```

**التقييم:**
- ✅ **الإيجابيات**: practical learning، تدريب فعلي
- ✅ **يدعم الهدف الخامس**: "التدريب على التعامل"
- 🎯 **الأولوية**: عالية (يمكن تنفيذها قريباً)

**التنفيذ المقترح:**
- نستخدم الأعراض اللي المستخدم اختارها
- نولّد سيناريوهات مخصصة بناءً عليها
- مثلاً: لو اختار "بتحس بالذنب"، نديله تدريب على "رفض الطلبات بدون ذنب"

---

### 4. استخدام الأعراض في توليد الخطة - حاسم!

**الفكرة:**
الخطة الديناميكية تستخدم الأعراض اللي المستخدم اختارها.

**مثال:**
```typescript
// في dynamicPlanGenerator.ts

function generatePersonalizedPlan(symptoms: string[]) {
  const plan = [];
  
  // لو المستخدم اختار "guilt" (الذنب)
  if (symptoms.includes('guilt')) {
    plan.push({
      week: 1,
      title: "تحدي الذنب المفتعل",
      actions: [
        {
          type: "reflection",
          text: "اكتب 3 مواقف حسيت فيها بذنب مش مستحق",
          helpText: "مثال: لما رفضت طلب مش معقول"
        },
        {
          type: "practice",
          text: "قول 'لأ' مرة واحدة هالأسبوع بدون تبرير",
          helpText: "مثلاً: 'مش هقدر، شكراً' - وبس!"
        }
      ]
    });
  }
  
  // لو المستخدم اختار "exhausted" (الإرهاق)
  if (symptoms.includes('exhausted')) {
    plan.push({
      week: 2,
      title: "حماية الطاقة",
      actions: [
        {
          type: "observation",
          text: "راقب مستوى طاقتك قبل وبعد اللقاء",
          helpText: "من 1 لـ 10، كنت قد إيه نشيط؟"
        },
        {
          type: "practice",
          text: "حدد وقت اللقاء (30 دقيقة فقط)",
          helpText: "لما الوقت يخلص، اعتذر وانصرف"
        }
      ]
    });
  }
  
  return plan;
}
```

**التقييم:**
- ✅ **الإيجابيات**: خطة دقيقة جداً، personalization 100%
- ✅ **يحقق الهدف السادس**: "التعافي من كل العلاقات اللي أثرت عليه"
- 🎯 **الأولوية**: **عالية جداً** (يجب تنفيذها!)

### 5. Severity Scale (مقياس الشدة) - مستقبلاً

**الفكرة:**
بدل checkbox بسيطة، نضيف مقياس:

```
☐ ما بحسش بيه
☐ أحياناً (1-2 مرة في الشهر)
☐ غالباً (مرة في الأسبوع)
☐ دايماً (يومياً)
```

**التقييم:**
- ✅ **الإيجابيات**: data أدق، نقدر نتتبع التحسن بدقة
- ⚠️ **السلبيات**: أطول، قد يسبب friction
- 🎯 **الأولوية**: منخفضة (v2.0)

### 6. مراجعة الأعراض بعد شهر - مهمة!

**الفكرة:**
بعد شهر من استخدام الخطة، نطلب من المستخدم يراجع الأعراض.

```
📊 مراجعة التقدم

قبل شهر، كنت بتحس بـ:
✓ الذنب لما تقول 'لأ'
✓ الإرهاق بعد اللقاء
✓ التفكير لساعات

دلوقتي، لسه بتحس بنفس الحاجات؟
☐ الذنب لما تقول 'لأ' (✓ اتحسن | ☐ زي ما هو | ☐ زاد)
☐ الإرهاق بعد اللقاء
☐ التفكير لساعات

💡 النتيجة: تحسنت في 2 من 3 أعراض - مبروك!
```

**التقييم:**
- ✅ **الإيجابيات**: progress tracking، motivation، proof of improvement
- 🎯 **الأولوية**: متوسطة-عالية
