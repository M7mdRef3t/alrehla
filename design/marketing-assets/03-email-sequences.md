# Email Sequences

## Sequence Goal
Help the user understand the product, activate quickly, and return when needed.

## Day 0 - Welcome
Subject: Welcome to the journey
Body: Thanks for joining. In a few minutes, you will start organizing relationships in a clearer and easier way.
CTA: Get started

## Day 2 - First Value
Subject: Your first real value
Body: Add a few names and see how they turn into a clear picture instead of a scattered list.
CTA: Open the map

## Day 5 - Social Proof
Subject: How others use it
Body: Many users discover connections they did not notice before just by arranging names in one map.
CTA: See the example

## Day 10 - Re-engagement
Subject: Want to continue?
Body: If you paused at any step, you can come back and keep going with a clearer picture.
CTA: Continue the journey
