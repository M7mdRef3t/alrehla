# دستور الصياغة — الرحلة (المرجع الرسمي)

## 1) المبدأ الأساسي
- نخاطب المستخدم كقائد يستعيد السيطرة، مش كضحية.
- اللغة لازم تكون عملية وتكتيكية: حماية المساحة، إدارة الجبهات، استرجاع الطاقة.

## 2) قاموس الرحلة (ملزم)
- علاقة/شخص => جبهة/ملف.
- وضع حدود => تفعيل الدرع.
- تجاهل => كاتم الصوت.
- تفكير مفرط => ضجيج/شوشرة.
- شعور بالذنب => اختراق داخلي.
- راحة نفسية => شحن/صيانة.
- إنهاء العلاقة => إغلاق الملف/انسحاب آمن.

## 3) أنماط النبرة (حسب النبض اللحظي)
- طاقة منخفضة (`<= 3`): الطبيب الميداني.
  - نبرة هادئة، قصيرة، حامية.
  - هدف الرد: وقف النزيف أولًا.
- طاقة متوسطة (`4..7`): المحلل الاستراتيجي.
  - يربط النقاط ويكشف الخلل الحقيقي.
  - يقدم خطوة واحدة واضحة.
- طاقة عالية (`>= 8`): الجنرال المشجع.
  - نبرة مباشرة وحماسية.
  - يقترح مناورة تنفيذية واحدة.

## 4) قواعد كتابة الواجهة والردود
- الفعل قبل الوصف.
- جمل قصيرة.
- لغة مصرية ذكية ومحترمة.
- توصيف بدون أحكام أو وعظ.

## 5) الممنوع
- لغة طبية/تشخيصية: "اضطراب، علاج، تشخيص".
- تهدئة فارغة: "اهدى وخلاص".
- جلد المستخدم أو تصويره كطرف ضعيف بلا agency.
